# ethfaucet
Simple ETH faucet built with Node.

### Install
```
npm install
```

### Running
Start MongoDB (Windows):
```
C:\Program Files\MongoDB\Server\3.2\bin\mongod.exe
```

Start the app:
```
npm start
```

### Testing
```
mocha
```

### Coverage
```
npm run coverage
```

### Linting
```
npm run linter
```
